## Image Guidelines

- **Format**: JPG or PNG
- **Optimization**: Compress images for web (use tools like TinyPNG)
- **Naming**: Use lowercase with hyphens (e.g., `hero-device.jpg`)
- **Size**: Keep images under 500KB when possible
- **Dimensions**: 
  - Hero image: 1200x800px
  - Other images: Vary based on content, but optimize for web

## Placeholder Images

The website will show placeholder boxes until you add the actual images. Replace placeholders by:
1. Adding images to this folder
2. Ensuring filenames match those referenced in `index.html`
3. The placeholders will automatically hide when images load
